Запускается из main.py, для запуска требуется PyQt5 и PyQt5-sip.
Интерфейс. Запуск:
Server - название сервера
Channel - название канала
Username - Имя пользователя
Для присоединения нажать Connect.

Debug - для отслеживания сообщений о состоянии соединения и сообщений сервера.
